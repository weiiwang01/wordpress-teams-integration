OpenID teams implementation for Wordpress

== Description ==

This plugin implements OpenID teams in Wordpress.

== Installation ==

This plugin follows the [standard WordPress installation method][]:

1. Upload the code to a folder named `openid-teams` in the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure the plugin through the 'Launchpad' section of the 'Options' menu

[standard WordPress installation method]: http://codex.wordpress.org/Managing_Plugins#Installing_Plugins
